# 文字轉語音小幫手(使用macOS內建庫) 說明檔

使用基於 macOS "Say"指令的文字轉語音應用程式，使用 SwiftUI 建構。（本說明檔由Guanzi(Ai)生成）

## 功能特色

- 📝 文字轉語音合成
- 🌍 支援多種系統內建語言和語音
- 🎧 試聽語音輸出
- 💾 匯出音訊檔案（AIFF、WAV、M4A 格式）
- ⚡ 可調節語速（25%-250%）
- 📂 讀取和儲存文字檔案
- 🔤 可調節字體大小
- 🔢 批次匯出計數功能

## 系統需求

- macOS 12.0 或更高版本
- Xcode 命令列工具（用於建構）

## 使用方法

1. 啟動應用程式
2. 在文字方塊中輸入或貼上文字/讀取TXT文字檔
3. 選擇語言和講述人
4. 點擊「試聽」按鈕預聽
5. 點擊「匯出音檔」按鈕儲存音訊檔案

## 技術架構

- SwiftUI - 使用者介面
- AVFoundation - 語音合成
- macOS 內建語音庫
